\# ShopEZ - E-Commerce Application



\## Tech Stack

\- React.js

\- Node.js

\- Express.js

\- MongoDB

\- JWT Authentication

\- Bootstrap



\## Features

\- User Registration \& Login

\- JWT Authentication

\- Product Listing

\- Product Details

\- Search \& Category Filter

\- Shopping Cart

\- Wishlist

\- Checkout

\- Order Management

\- Stock Deduction

\- Admin Dashboard

\- Analytics Dashboard



\## Installation



\### Backend

cd server

npm install

npm run dev



\### Frontend

cd client

npm install

npm run dev
